# Wheather-App

A simple Weather Application built using HTML, CSS, and JavaScript that allows users to check the current weather of any city in the world.

The app fetches real-time weather data using a weather API and displays information such as temperature, humidity, and weather conditions.

 Live Demo

You can view the live project here:

https://karthik-2376.github.io/Weather-App/

Features
Search weather by city name
Displays current temperature
Shows humidity level
Displays weather condition icons
Fast and responsive interface
Technologies Used

HTML – Structure of the application

CSS – Styling and layout

JavaScript – Logic and API integration

Weather API – Fetch real-time weather data
